> *"Not all those who wander are lost."*
> — J.R.R. Tolkien

So I created a blog. Truth is, now that AI can handle part of what I am doing, I can spend more time on side quests. And since I am more rapidly side questing than ever, there is something to blog about in the first place.

This blog is an application written using Claude Code, the one that spins up a cloud container and writes code on demand. I wrote the instructions (CLAUDE.md) first using chat (it's faster), then Claude Code went on and did its thing. For flexibility I added the option to just edit the .md files or have an AI agent post it straight away using the MCP surface.

Also, I introduced an editorial agent, who by contract is obliged to insert a matching meme into every post.

![RPG quest log: the main quest "day job" is partially delegated to AI, while 47 side quests are active — including building this blog and writing this post](/static/memes/so-i-created-a-blog.svg)

So welcome, and stay tuned. I'll be writing about whatever I feel like sharing.
