Welcome to the blog. This is a starter post so the site has content on first deploy — feel free to edit or delete it.

This blog keeps **all content as plain Markdown files** under `posts/<year>/<slug>.md`, committed to Git. The app is just a viewer over those files, so the content outlives any particular tool.

Authoring is simple: edit a file locally, commit, push, and the deploy updates automatically.
